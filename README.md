# Política de Privacidade – DormiBem

Última atualização: 19/12/2025

O aplicativo **DormiBem** respeita a privacidade dos seus usuários e está comprometido em proteger qualquer informação utilizada durante o uso do aplicativo.

## 1. Coleta de informações
O aplicativo DormiBem **não coleta, armazena ou compartilha dados pessoais** dos usuários.

## 2. Uso de informações
Nenhuma informação é utilizada para fins comerciais, analíticos ou publicitários.

## 3. Permissões
O aplicativo pode solicitar permissão de áudio apenas para reprodução dos sons.

## 4. Compartilhamento
Nenhum dado é compartilhado com terceiros.

## 5. Anúncios
Atualmente, o aplicativo não exibe anúncios personalizados.

## 6. Público-alvo
O aplicativo é destinado ao público adulto.

## 7. Alterações
Esta política pode ser atualizada a qualquer momento.
